# 정각 알림 프로그램

매 시간 정각에 알림음을 재생하고 Windows 토스트 알림을 표시하는 프로그램입니다.

## 주요 기능

1. **GUI 인터페이스**: 사용하기 쉬운 그래픽 사용자 인터페이스
2. **시스템 트레이 최소화**: 백그라운드에서 실행 가능
3. **자동 시작 설정**: 컴퓨터 시작 시 자동 실행 On/Off
4. **커스텀 알림음**: 기본 Ring10.wav 파일 또는 다른 오디오 파일 선택 가능
5. **토스트 알림**: 정각마다 "HH시 정각입니다" 메시지 표시 (3초 후 자동 사라짐)

## 설치 방법

1. 필요한 패키지 설치:
```bash
pip install -r requirements.txt
```

2. Ring10.wav 파일을 프로그램과 같은 폴더에 배치
   (없으면 다른 오디오 파일을 선택할 수 있습니다)

## 사용 방법

1. 프로그램 실행:
```bash
python HourlyAlram.py
```

2. GUI에서 설정:
   - "정각 알림 활성화" 체크박스로 알림 On/Off
   - "컴퓨터 시작 시 자동 실행" 체크박스로 자동 시작 설정
   - "사운드 변경" 버튼으로 알림음 변경
   - "사운드 테스트" 버튼으로 미리 듣기
   - "시스템 트레이로 최소화" 버튼으로 백그라운드 실행

3. 시스템 트레이에서:
   - 아이콘 클릭으로 "열기" 또는 "종료" 선택

## 요구사항

- Python 3.7 이상
- Windows 10/11
- 오디오 파일 (WAV, MP3, OGG 형식 지원)
